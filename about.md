# ModifyLoadingLayer

## Overview

This mod allows you to modify the loading screen however you want and share it. this includes the following:
- Moving elements
- Scaling elements
- what else
- Set opacity
- change elements' sprites (broken a little as of now.)
- add your own sprites!
